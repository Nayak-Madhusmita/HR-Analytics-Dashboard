📊 HR Analytics Dashboard (Tableau)
🧾 Project Overview

The HR Analytics Dashboard is an interactive data visualization project developed using Tableau to analyze workforce trends, employee attrition, and organizational performance. The dashboard helps HR teams and decision-makers monitor key metrics and identify patterns affecting employee retention and satisfaction.

This project demonstrates skills in:

Data cleaning and preparation
Dashboard design and storytelling
KPI tracking and HR data analysis
🎯 Objectives
Analyze employee attrition and retention patterns
Monitor workforce demographics and departmental distribution
Identify key factors influencing employee turnover
Provide actionable insights to support HR decision-making
🛠️ Tools & Technologies
Tableau Desktop – Dashboard development
Microsoft Excel – Data storage and preprocessing
Data Visualization & Analytics – KPI and trend analysis

📊 Dashboard Features
1. Employee Overview
Total employees
Attrition count and attrition rate
Average age and average salary
2. Attrition Analysis
Attrition by Department
Attrition by Gender
Attrition by Age Group
Attrition by Education Field
3. Job Role Insights
Employee distribution across job roles
Identification of roles with highest turnover
4. Interactive Filtering

Users can dynamically filter the dashboard by:

Department
Gender
Education
Age group
📸 Dashboard Preview
🔍 Key Insights Derived
Certain departments show significantly higher attrition rates.
Younger employees demonstrate higher turnover compared to senior staff.
Job roles with lower satisfaction scores correlate with higher attrition.
